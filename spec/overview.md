# 项目概述

## 项目定位

Stuwise 是一个面向学生日常场景的 AI Agent，通过浏览器访问，充当私人助理角色。用户只需配置 API Key 即可开箱即用。

## 设计理念

- **零配置上手**：输入 API Key，启动，浏览器打开，即可使用
- **单用户本地**：一人一机，数据完全本地，不做多用户架构
- **通用工具 + 场景 Skill**：工具层提供通用能力（文件操作、网络搜索），场景特化通过 Skill 实现（笔记管理、知识库、排课表等），Skill 从本地 `skills/` 目录加载
- **渐进增强**：从最小可用的聊天开始，逐步增加 Skill、知识库等功能

## 核心交互流程

```
打开浏览器 → 看到聊天界面 → 输入问题 → Agent 流式回复
                                          ├── 调用通用工具（read/bash/edit/write）
                                          ├── 或调用网络工具（web_search/web_fetch）
                                          └── 流式展示回复 + 工具执行状态
```
