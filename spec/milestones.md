## M0 目标

> 工程骨架搭建 + 端到端最小可用原型。
> 能启动服务器 → 浏览器打开聊天界面 → 发送消息 → Agent 流式回复（纯文本对话，不含工具和 Skill）。

任务拆分：
- M0-1 项目初始化 + 目录结构 + API Key 配置
- M0-2 AgentHarness 工厂函数 + 单元/集成测试
- M0-3 Hono 服务器 + API 路由 + Vite 开发代理
- M0-4 Vue 前端骨架 + 聊天 UI（mock 数据）
- M0-5 SSE 事件推流 + 前端状态管理
- M0-6 端到端集成验证

完成后可交付：
- `npm run dev` 启动开发服务器
- 浏览器打开看到聊天界面
- 配置 API Key 后能纯文本对话
- 对话历史持久化到 JSONL
