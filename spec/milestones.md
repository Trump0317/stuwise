## M0 目标

> 工程骨架搭建 + 端到端最小可用原型。
> 能启动服务器 → 浏览器打开聊天界面 → 发送消息 → Agent 流式回复（纯文本对话，不含工具）。

完成后可交付：
- `npm run dev` 启动开发服务器
- 浏览器打开看到聊天界面
- 配置 API Key 后能纯文本对话
- 对话历史持久化到 JSONL
